# To-Do List Web Application

## Overview
This project is a simple interactive To-Do List web application built using HTML, CSS, and JavaScript. It allows users to add, manage, and delete tasks dynamically in the browser.

The goal of this project is to demonstrate fundamental web development skills, including DOM manipulation, event handling, and basic UI design.

---

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed (toggle line-through effect)
- Interactive user interface
- Real-time updates without page refresh

---

## Technologies Used
- HTML5 (structure)
- CSS3 (styling)
- JavaScript (functionality & DOM manipulation)

---

## How It Works
1. The user types a task into the input field.
2. Clicking the "Add" button creates a new list item.
3. Each task can be:
   - Clicked to mark as completed
   - Deleted using the "Delete" button
4. All updates happen instantly in the browser using JavaScript.

- Understanding DOM manipulation
- Working with events in JavaScript
- Creating dynamic web applications
